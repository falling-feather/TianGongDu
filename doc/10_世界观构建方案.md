# 《天工渡 / Heavenwrights》世界观构建方案

> 版本：0.5  
> 日期：2026-06-23  
> 状态：世界观底层框架草案，已调整为承平社会与地方课题基调  
> 目标：让工艺经营、类 2.5D 战斗、地区包更新和服务器运营内容自然处在同一套世界逻辑中，而不是靠抽象概念硬连。

## 1. 构建原则

《天工渡》的世界观不从一个新造概念开始，而从中国语境里本来就合理存在的社会结构开始：

- 朝廷官造与工部制度。
- 地方行会与师徒传承。
- 宗族、祠庙、祖师信仰。
- 商路、票号、货栈、镖行。
- 山川、水利、桥梁、窑炉、作坊。
- 器物有灵、久物成精、祟、魇、镇物、厌胜等民间想象。

这些东西本来就能解释经营、战斗和探索：

- 经营来自工坊、材料、订单、行会、商路和民生。
- 战斗来自器物、场所、记忆、祠社和人事纠纷积累出的祟乱。
- 成长来自拜师、工籍、谱牒、试艺、行脚和器具返修。
- 地区包来自不同地方的产业、民俗、地貌和权力关系。

## 2. 世界基底

这是一个架空九州王朝的承平世界。它不需要直接绑定某个真实朝代，但气质可以参考唐宋明之间的混合：城市成熟，商路通达，行会活跃，官造体系仍有威信，地方工坊各有名声，文人消费、节令礼俗、宗族祠庙和远行贸易共同构成日常生活。

这个世界不是末世，也不是乱世。大多数地方稳定、富足、有秩序，人们愿意修桥、造船、办灯市、开窑会、拜祖师、赶集、订货、送学徒。也正因为社会整体安定，技艺才有精进、竞赛、流通和创新的空间。

朝廷设有管理百工、官造、贡作和大型工事的机构，暂名“天工院”。天工院不是神秘魔法组织，而是一个兼具官造档案、工籍谱牒、工程验收、祖师祭仪和异事处置职责的庞杂机构。

天工院与地方社会并非对立，而是互相依存：

- 京城工署保存档案、标准和大工程经验。
- 地方工局协调材料、工坊和公共工事。
- 行会维护师徒规矩、技艺名声和市场秩序。
- 商帮连接远方订单、货栈、船队和票号。
- 祠庙承接祖师祭仪、节令活动和地方信任。
- 民间匠人掌握手上工夫，也不断改良器具与做法。

玩家所在的“天工渡”是天工院与地方行会共同承认的一种行脚制度。渡人持有渡牒，能在官局、行会、工坊、商路和祠庙之间通行，既会手艺，也懂规矩，负责协助地方处理那些普通工坊难以单独解决的器事、工事、商事和异事。

## 3. “渡”的含义

“渡”不要解释成超能力，而是多重身份：

- 渡河过境：行走九州。
- 渡牒通行：有资格进入工坊、仓库、工地、祠社和禁地。
- 渡人解结：在官、商、匠、民、祠之间调停。
- 渡器归位：让器物重新回到应有用途。
- 渡艺传承：把将断的技艺重新接上人。

这样“天工渡人”的身份自然成立：玩家不是侠客套皮，也不是抽象修复者，而是一个拥有工籍、师承、行脚资格和处理异事经验的特殊匠行人。

## 4. 主线驱动：执灯渡人与《天工录》

项目需要一条能持续驱动玩家行走九州的世界线。当前建议将主线定为：

> 玩家是天工院“巡灯司”下的年轻执灯渡人，持临时渡牒与一盏工灯行脚九州。每到一地，不只是解决局部事件，而是在检校地方工印、见证工评盛事、补录当代百工的新做法、新师承、新器物和新祟局，也追寻自己师承留下的未竟线索，最终从候补渡人成为真正能独当一面的执灯渡人。

这里可以借鉴“秉烛人”式的结构，但不照搬名词和对象：

- 秉烛人的核心是以正式职分巡行、记录、监管特殊存在。
- 《天工渡》对应为执灯渡人以渡牒和工灯巡行九州，检校地方工印与工事异象。
- 秉烛人面对的是巨兽与代理人问题；执灯渡人面对的是工印、工事、器物、师承和祟局。
- 秉烛人的武艺是职责的一部分；执灯渡人的战斗、身法和门径也必须从第一天就是职分能力，而不是进入祟局才突然发生。

这条主线同时提供三重驱动力。

### 4.1 个人驱动

玩家开局不是预言中的唯一人物，而是一个刚取得临时渡牒的年轻执灯渡人。玩家的师傅曾参与《天工录》续修，负责记录九州各地正在变化的工艺生态，也曾承担巡灯司的外勤检校。

师傅没有留下“拯救世界”的遗命，而是留下了一卷没有写完的行脚录和一盏没有完全校准的工灯：

- 有些页只写了地名。
- 有些页夹着材料样本。
- 有些页盖着未解释的工印。
- 有些页记录了奇怪的祟局形态。
- 有些页写着师傅对某位匠人、某场工评、某座桥、某件器物的评价。

玩家出发的第一目标很清楚：走完师傅没走完的路，弄明白这些空白页为什么没有完成，校准工灯上缺失的几道工印，并把自己的名字正式写进渡牒。

### 4.2 世界驱动

天工院正在主持新一轮《天工录》续修。

这不是因为世界出了大问题，而是因为承平年代百工兴盛，旧档案已经不足以记录新的现实：

- 新商路让地方工艺传到更远处。
- 新材料和新工具改变了旧工序。
- 年轻匠人不断提出改良做法。
- 行会、官局、商帮都需要新的工艺标准。
- 各地祠社、节庆和工评也在形成新的礼俗。

《天工录》因此不是静态百科，而是“活的工艺档案”。谁能入录，怎样入录，代表一种技艺在九州中的地位、信誉和未来。

玩家每完成一个地区，不只是拿奖励，而是为《天工录》补上一条活页：

```text
地区见闻
-> 师承记录
-> 工序记录
-> 器具特质
-> 祟局案例
-> 地区选择
-> 工印落款
```

### 4.3 地区驱动

每个地区都能自然成为一章。

地区不是“待拯救的病灶”，而是有自己的盛事、课题和机会：

- 江南雨巷：试伞会与远方伞单。
- 龙泉山坊：开炉试剑与青瓷火候。
- 徽州文房：纸墨工评与商帮订单。
- 巴蜀竹乡：竹机改良与山路货运。
- 西域巴扎：玉作、乐舞和远行商队。

玩家到一个地区的理由可以很自然：

- 天工院派发补录任务。
- 师傅旧页留下线索。
- 行会邀请渡人见证工评。
- 商帮请求调停货约。
- 祠社准备节庆，需要检校器物。
- 某位名匠愿意授艺，但要求玩家通过试炼。

这样，长期维护时每个地区包都能接入主线，而不需要每次都发明一场大灾。

### 4.4 暗线驱动

《天工录》续修过程中，玩家逐渐发现：不同地区的祟局虽然来源各异，却反复出现同一种“无款工印”。工灯会在靠近这类印记时出现异常反应，这让“无款工印”从背景谜题变成可探索、可追踪、可战斗验证的长期线索。

这枚工印不代表幕后黑手，也不代表世界即将倾覆。它更像一个历史谜题：

- 它可能来自天工院初修《天工录》时遗漏的一支工门。
- 也可能来自一位不愿留名的匠宗。
- 也可能是某种古老工法在不同地区留下的共同痕迹。
- 还可能与玩家师傅未完成的行脚有关。

暗线的作用不是把所有地区变成同一阴谋，而是给玩家持续的好奇心：为什么这些相隔很远的地方，会在器物、节令、祟局或工印上产生微妙呼应？

## 5. 执灯渡人的职分能力

为了解决“社交、经营、战斗割裂”的问题，执灯渡人的战斗能力必须从世界观和时间线里自然成立。

执灯渡人不是到了祟局才战斗。他们的日常职责本来就包括：

- 护送材料和样器。
- 巡检桥梁、窑口、工坊和货栈。
- 处理路途中出现的游祟、器影和失控机关。
- 在工评、试艺、开窑、赛伞等场合见证并参与技艺比试。
- 用身法和门径进入普通人到不了的工事位置。
- 必要时保护匠人、商队、学徒和地方文书。

因此玩家的基础战斗能力来自“执灯训练”：

| 职分训练 | 游戏能力 |
| --- | --- |
| 行脚身法 | 平面移动、闪避、短距离越行、攀越 / 翻越等地形能力 |
| 护牒刀法 / 短兵 | 基础轻击、重击、破招 |
| 工灯照形 | 显示弱点、识别机关、照出器影 |
| 定印手 | 格挡、弹反、稳定祟局 |
| 试艺规程 | 切磋、工坊挑战、Boss 阶段教学 |
| 归器礼 | 战斗终结后的归位交互 |

地方门径不是从零开始学一套游戏，而是在执灯训练之上加入当地技艺。例如纸伞门径并不是玩家突然学会战斗，而是把已有的格挡、身法、照形、定印能力，转化为架伞、弹反、收风、借风越行。

## 6. 时间线组织

每个地区的时间线要保证行动连续，而不是“先社交经营，最后突然进副本打架”。

推荐地区时间线：

```text
抵达地区
-> 路上遭遇小型游祟 / 护送 / 试艺，展示执灯基础战斗
-> 进入市井层，拜访官局、行会、工坊、祠社
-> 用工灯检校材料、账目、工印或器物痕迹
-> 学习当地门径的第一段用法，并立刻在路面战斗或工事关卡中使用
-> 经营材料、订单和工坊关系
-> 多次穿插小战斗、切磋、机关和采集
-> 祟局显形，Boss 作为此前所有行动的汇总检验
-> 归位、落款、入录，获得下一处线索
```

这样战斗、社交、经营、学习和祟局不是分段拼接，而是一条连续行脚。

## 7. 主线循环

主线循环建议固定为：

```text
获得《天工录》线索或工灯反应
-> 抵达地区
-> 路途小战斗或护送事件
-> 拜访官局 / 行会 / 工坊 / 祠社
-> 参与当地盛事或工艺课题
-> 学习门径或工艺
-> 经营材料、订单、工坊关系
-> 在工事层多次验证新能力
-> 进入工事层和祟局层验证技艺
-> 完成地区选择
-> 《天工录》落款，获得下一处线索
```

这条循环的重点是“记录、见证、精进”，不是“巡回救火”。

## 8. 超自然边界

本世界不需要宣布“工艺维持世界秩序”。更自然的说法是：

> 器物、工事和技艺长期嵌在人们的生活、信仰、交易和记忆里。它们承载的用处、规矩、心愿和地方记忆越深，就越容易在关键节点显出异象。大多数异象并非灾祸，而是提醒、试炼、回响或尚未安放妥当的余波。

这里的“祟”不是怪物种族，而是人、器、地、事之间纠缠出来的异象。

祟局的来源：

- 器物：伞、剑、瓷、漆、纸、墨、玉、乐器、戏具。
- 工事：桥、窑、渠、矿、船坞、城楼、戏台。
- 账约：急单、贡作、远方订单、货期变动、价格协商。
- 师承：新旧工法分歧、学徒试艺、名师较艺、谱牒考证。
- 祠社：祖师祭、开窑会、灯市、社戏、桥成礼。
- 民生：节令需求、婚嫁器用、修补日用、地方新风尚。

战斗因此不是“打文化怪”。它更像进入一个具体事件的祟局，面对被放大的技艺难题、器物性情、地方记忆或人事结扣。玩家通过战斗让异象平复，再回到现实层完成工序、交付、调解、修复或庆成。

## 9. 三层空间

每个地区可以按三层组织。

### 9.1 市井层

玩家经营、对话、接单、买卖、采集、修复、跑商、培养关系的地方。

典型场景：

- 工坊。
- 夜市。
- 货栈。
- 码头。
- 师傅家。
- 行会馆。
- 祠庙前街。

### 9.2 工事层

类 2.5D 动作区域的主要来源。它不是抽象副本，而是地方产业真实依赖的空间。

典型场景：

- 竹林。
- 纸坊。
- 窑场。
- 旧桥。
- 水渠。
- 矿洞。
- 戏台后台。
- 染坊晾场。

### 9.3 祟局层

Boss 和强战斗发生的地方。它不是另一个世界，而是现实场所被器物、记忆、节令、祭仪、工序和人事牵动后显出的异象状态。

同一座桥，在市井层是交通；在工事层是修复对象；在祟局层可能变成断桥雨幕 Boss 场。

## 10. 经营与战斗的自然关系

经营不是基地菜单，战斗也不是外插副本。它们是一条自然流程：

```text
地区迎来新机会或新课题
-> 材料、订单、师承、商路、民生或祠社出现需要协调的节点
-> 玩家通过经营、调查和学艺理解地方脉络
-> 课题在工事层和祟局层外化为敌人、机关、Boss
-> 战斗平复异象，验证新技艺
-> 玩家用工艺、资源和选择完成交付、修复、调解或庆成
-> 地区进入更成熟的新状态
```

如果玩家只打 Boss，不处理订单、材料或师承，地区只是暂时平静。  
如果玩家只经营，不面对祟局，关键技艺和器物性情也得不到真正验证。

## 11. 四维系统的世界内表达

“传承 / 市场 / 生活 / 精神”是策划层表达。世界内不要让 NPC 直接说这四个词像系统数值。

在世界内，它们可以表现为“工牒四目”：

| 策划维度 | 世界内表达 | 具体含义 |
| --- | --- | --- |
| 传承 | 师谱 | 师徒、谱牒、工序、祖法 |
| 市场 | 货账 | 订单、价格、商路、货期 |
| 生活 | 民用 | 居民日用、修补、节令需求 |
| 精神 | 礼信 | 祖师、祠社、禁忌、仪式意义 |

玩家看到的 UI 可以仍然清晰，但剧情中应尽量通过账簿、师傅评价、货栈账目、居民反馈和祠庙变化来表达。

## 12. 江南雨巷落地

雨巷镇不是因为“纸伞工艺失序”而抽象出事，而是一个繁荣水镇遇到新机会后产生的自然课题：

1. 雨巷镇靠纸伞、染纸、水路集市和灯市闻名，行人多，店铺旺，学徒也愿意来学。
2. 今年雨季将至，远方货栈、码头客商和本地灯市同时追加订单，镇上第一次接到这么大的伞单。
3. 年轻作坊希望改良工序、提高产量；老伞师担心赶工会失了手感和口碑。
4. 行会决定办一场“试伞会”，让各家作坊在雨巷风口展示新伞，也借机评定今年的货单分配。
5. 玩家来到雨巷镇，既要学伞法，也要帮师傅选竹、糊纸、试伞，理解为什么一把伞不能只看能不能挡雨。
6. 试伞会当夜，旧桥雨棚下积存的旧伞、风声、灯影和众人的期待牵动出“破伞祟局”。它不是灾难，而像一场被放大的试艺：不懂伞骨、伞面、风口和开合节奏的人过不了。
7. 玩家抵达雨巷前，先在水路护送一批伞骨和桐油，遭遇雨中游祟，展示执灯基础战斗。
8. 入镇后，玩家在市井层处理订单和关系，在竹林与旧桥等工事层多次使用架伞、照形和收风，最后进入断桥雨幕验证纸伞门径。
9. Boss 平复后，玩家决定这批订单如何分配、是否采用改良工序、如何保留师傅的试伞规矩，以及夜市灯会如何呈现雨巷纸伞。

这样，纸伞门径、工艺经营、夜市选择、类 2.5D 动作区域和 Boss 战是同一事件的不同层面。

## 13. 长期主线

长期主线不依赖单一终极反派。更适合的是“玩家以执灯渡人的职分补完《天工录》、追寻师承旧页、参与九州百工盛事，并逐渐理解无款工印背后的历史呼应”。

大主线围绕四件事推进：

- 《天工录》续修：天工院正在重修百工档案，邀请渡人行走各地，记录新工法、地方名匠、异象案例和可传之艺。
- 师傅旧页：玩家不断补全师傅留下的空白页和未解释工印。
- 九州工评：各地工坊、行会和官局定期举行工评、开窑、灯会、赛伞、试剑、竞织等盛事，玩家参与其中，推动地区技艺精进。
- 行路见闻：玩家在不同地区看到技艺如何进入市场、生活、礼俗和个人志向，逐渐形成自己的门径组合。
- 无款工印：不同地区出现的微妙共同痕迹，推动中长期剧情展开。

玩家每到一个地区，不是简单拯救当地，而是参与当地正在发生的选择：

- 古法如何与新工法相容。
- 名匠规矩如何传给年轻学徒。
- 好器物如何既能售卖，也能留在百姓日用里。
- 节庆、祠社和演出如何既热闹，又不失礼信。
- 地方如何通过自己的技艺获得更好的名声。

## 14. 风格边界

避免：

- 直接说“工艺是魔法”。
- 直接说“世界靠工艺维持”。
- 把非遗项目做成怪物。
- 把每个地区写成同一种道德题。
- 把玩家写成无所不能的中心人物。
- 把九州写成处处只有问题、缺少日常欢欣。
- 把《天工录》写成单纯收集图鉴。
- 把无款工印写成俗套阴谋标记。
- 把战斗只放在祟局里，导致玩家日常行脚缺少行动感。

坚持：

- 先有地方生活，再有异事。
- 先有稳定秩序，再有局部课题。
- 先有真实工序，再有玩法抽象。
- 先有人的选择，再有 Boss 的形态。
- 打完以后必须回到现实生活和技艺精进。
- 每个地区都要给《天工录》增加一条有意义的记录。
- 主线暗线要提供好奇心，而不是压垮地区自身主题。
- 玩家进入地区的第一小时内，就要让执灯职分、战斗能力、工艺学习和地方关系同时发生。

## 15. 一句话版本

《天工渡》的世界不是“工艺魔法世界”，而是一个架空九州的承平社会：官造有序、行会兴盛、商路通达、祠社热闹、师承活跃。玩家作为持临时渡牒和工灯的年轻执灯渡人，带着一卷未完成的《天工录》行走各地，在护送、巡检、试艺、经营、战斗和祟局归位中补录师承、器物、工印与地方盛事，在不断精进中追寻师傅旧页留下的线索，并让器物、场所和人各得其位。
