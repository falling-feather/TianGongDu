# 《天工渡 / Heavenwrights》资料索引与调研计划

> 版本：0.3  
> 日期：2026-06-23  
> 状态：当前有效资料索引  
> 目的：记录外部依据、已采用结论和后续调研任务。

## 1. 技术资料

当前主线：

- Axmol：`https://axmol.dev/`
- Axmol GitHub：`https://github.com/axmolengine/axmol`

采用结论：

- Axmol 作为 C++ 2D 主框架。
- Web 端为首要发布媒介，Axmol WebAssembly / WebGL2 链路必须在 M0 技术冒烟中验证。
- Python 作为内容工具链语言。
- 大型引擎编辑器不进入主线生产环境。
- 桌面端保留为开发调试、线下展示和未来封装方向。
- 《暖雪》作为像素、斜俯视动作平面、层次化场景和高反馈战斗的空间体感参考；不照搬其题材、叙事和数值结构。

## 2. 非遗资料

主要入口：

- 中国非物质文化遗产网：`https://www.ihchina.cn/`
- UNESCO 非物质文化遗产说明：`https://ich.unesco.org/en/what-is-intangible-heritage-00003`

采用原则：

- 非遗是活态实践、知识、技能、器物和场景，不是静态图鉴。
- 内容必须转化为行为、工艺、师承、场景和反馈。
- 敌人不是“文化符号怪物化”，而是失序现象具象化。

## 3. 旧项目参考

旧项目《百工归位》仅作为资料库和系统经验参考。

可继承：

- 地区、材料、NPC、订单、商路、四维系统。

不可照搬：

- 经营模拟主玩法。
- 表格驱动但缺少动作反馈的工艺体验。
- 一开始铺满全国大地图。

## 5. 首个纵切待补调研

### 5.1 江南纸伞

待查：

- 纸伞具体工序：选竹、削骨、穿线、糊纸、上油、晾晒、绘制。
- 伞骨数量、结构和常见缺陷。
- 油纸伞与雨、水、风的真实使用特征。
- 不同地区纸伞工艺差异。
- 代表性传承人、博物馆、地方资料。

玩法转译：

- 伞骨 -> 格挡稳定 / 破损。
- 伞面 -> 吃风 / 湿滞抗性。
- 开合 -> 弹反时机。
- 上油 -> 防雨 / 状态抗性。
- 绘伞 -> 外观、师承标记、精神维度。

### 5.2 江南水巷与雨

待查：

- 水巷空间结构、桥、埠头、沿街店铺。
- 雨季、潮湿、青苔、水面反光。
- 灯市 / 夜市是否适合作为第一地区支线氛围。

玩法转译：

- 水面波纹 -> 敌人攻击提示。
- 屋檐 / 桥 / 水面风口 -> 借风越行回访点。
- 雨幕 -> Boss 阶段机制。
- 夜市 -> 市场维度选择。

### 5.3 表达边界

待查：

- 哪些表达容易造成文化浅表化。
- 如何处理真实地域、传承人、工艺名称。
- 如何区分真实工艺和虚构门径。

原则：

- Boss 终点是归位，不是处决。
- 真实技艺与虚构玩法保持尊重和区隔。
- 所有文化资料进入游戏前要二次核验。
