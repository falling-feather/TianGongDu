# Heavenwrights Web Demo

> 文档定位：说明当前浏览器原型的运行方式、范围和操作。  
> 适用范围：`prototype/web-demo/` 的无外部依赖 Canvas 原型。  
> 最后更新：2026-06-29  
> 维护者：原型维护者  
> 关联文档：`../../doc/03_项目全局设计规划.md`、`../../doc/07_江南雨巷纵切规格.md`、`../../doc/12_测试与验收指南.md`

无外部依赖的浏览器原型，用于验证“江南雨巷 + 纸伞门径”的首屏体验。当前 M0.5 入口已补入封面、可探索地图、背包、NPC 行程、时辰推进、采集点和敌人预警。

运行：

```bash
npm run dev
```

打开：

```text
http://127.0.0.1:4174/
```

操作：

- `WASD` / 方向键：移动
- `Shift` / `Space`：冲刺
- `Q`：架伞
- `E`：收风
- `R`：借风越行
- `F`：交互 / 归位
- `M`：地图
- `I`：背包
- `N`：人物
- `T`：小憩推进时辰

验收路径见 `../../doc/12_测试与验收指南.md`。本 demo 是 M0 原型，不替代最终 Axmol/C++ 实现。
