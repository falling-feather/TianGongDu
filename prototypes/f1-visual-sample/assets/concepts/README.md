# 概念图资产记录

本目录保存通过内置图像生成流程产出的方向稿，并记录其用途：

- `environment-oblique-panorama.png`：整体环境、镜头和尺度确认。
- `combat-weight-impact.png`：战斗层次与打击重量确认。
- `gameplay-hud-screen.png`：完整游戏屏幕与 HUD 信息层级确认。

这些图像是项目内方向资产，不是可直接出货的角色立绘、贴图或 UI 切图。通过评审后再拆分正式生产需求。
