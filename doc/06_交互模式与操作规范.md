# 《天工渡 / Heavenwrights》交互模式与操作规范

> 版本：0.3  
> 日期：2026-06-23  
> 状态：当前有效交互草案  
> 适用：Web 端 2D / 类 2.5D 像素动作 RPG。

## 1. 交互原则

- 输入统一，门径变体。
- 少解释，多让玩家做。
- 战斗中不抢输入，非战斗中交互优先级稳定。
- 所有键位可重绑定。
- 手柄和键鼠体验都必须成立。

交互优先级：

1. 剧情关键交互。
2. NPC 对话。
3. 归位交互。
4. 工艺 / 修复。
5. 门 / 机关。
6. 采集。
7. 阅读 / 调查。
8. 拾取普通物品。

## 2. 推荐输入

手柄：

| 操作 | 按键 |
| --- | --- |
| 移动 | 左摇杆 / 十字键 |
| 闪避 / 取消 | B / Circle |
| 轻击 | X / Square |
| 重击 | Y / Triangle |
| 格挡 / 架伞 | LB / L1 |
| 技艺键 | RB / R1 |
| 门径技能 | RT / R2 |
| 工灯 / 照形 | LT / L2 |
| 交互 / 上下文动作 | A / Cross |
| 快捷道具 / 门径切换 | 方向键 |
| 菜单 | Menu |

键鼠：

| 操作 | 按键 |
| --- | --- |
| 移动 | WASD |
| 闪避 / 取消 | Left Shift / Space |
| 轻击 | 鼠标左键 |
| 重击 | 鼠标右键 |
| 格挡 / 架伞 | Q |
| 技艺键 | E |
| 门径技能 | R |
| 工灯 / 照形 | Tab / 鼠标中键 |
| 交互 | F |
| 快捷道具 | 1-4 |
| 菜单 | Esc |

基础移动按斜俯视动作平面处理，不以横向 A / D 和通用跳跃作为核心。跳跃、翻越、借风越行都属于地形上下文或门径能力，只有在明确提示、风口、屋檐落点或技能允许时触发。

## 3. 输入缓冲与取消

| 行为 | 缓冲 |
| --- | ---: |
| 轻击连段 | 180ms |
| 重击派生 | 150ms |
| 闪避 | 120ms |
| 格挡 | 100ms |
| 弹反 | 不缓冲，要求时机 |
| 交互 | 200ms，非战斗 |

取消规则：

- 受击 / 硬直不可随意取消。
- 闪避可取消轻击后摇，但不能取消所有重击前摇。
- 格挡可取消部分站立动作。
- 弹反只能从格挡准备或特定架势进入。
- 门径技能根据技能表单独定义。

## 4. 战斗交互

标准循环：

```text
观察敌人动作 -> 闪避 / 格挡 / 弹反 -> 制造破绽 -> 使用门径技能 -> 积累资源 -> 爆发或控场 -> 调整位置
```

伞师交互：

| 动作 | 输入 | 目标 |
| --- | --- | --- |
| 架伞 | 按住格挡 | 稳定防守，消耗气力 |
| 完美格挡 | 被击中前短窗口架伞 | 减少消耗，获得风息 |
| 弹反 | 精确格挡 / 格挡派生 | 打出破绽和反击窗口 |
| 收风 | 技艺键 | 聚拢、打断、吹散雨幕 |
| 伞旋 | 门径技能 | 范围控场 |
| 借风越行 | 门径技能 / 风口上下文 | 越过水面、风口、屋檐落点，完成探索回访与战斗换位 |

敌人提示：

- 普通攻击：动作前摇清楚。
- 可弹反攻击：武器、影子或雨线出现短促亮边。
- 不可格挡攻击：破伞纹或强提示。
- 环境攻击：地面水纹、风线、雨幕方向提示。

## 5. 工艺交互

工艺不做“全都 QTE”。每种工艺拆成：

```text
真实工序 -> 玩法抽象 -> 战斗 / 探索映射
```

纸伞示例：

| 工序 | 操作 | 映射 |
| --- | --- | --- |
| 采竹 | 观察竹节，选择材料 | 伞骨稳定，影响格挡消耗 |
| 削骨 | 控制力度区间 | 定力、破损概率 |
| 糊纸 | 控制覆盖与张力 | 吃风、湿滞抗性 |
| 试伞 | 在风口、水面和屋檐落点开合伞面 | 初始风息、借风稳定 |

工艺失败不简单重来，而是形成缺陷、师傅评价和返修空间。

## 6. NPC 交互

NPC 不只是菜单入口。至少承担一个功能身份和一个情感身份。

| 身份 | 作用 |
| --- | --- |
| 师傅 | 传授门径、技艺、试炼 |
| 匠人 | 修复、改造、识别材料 |
| 同行 | 切磋、协作、竞争 |
| 委托人 | 地区事件和订单选择 |
| 见证人 | 记录玩家行为并改变反馈 |

纵切关系值只保留：

- `trust` 信任。
- `tension` 分歧。

## 7. UI 与辅助

常驻 HUD：

- 生机。
- 气力。
- 风息。
- 当前技艺 / 技能。
- 交互提示。
- Boss 归位进度。

辅助设置：

- 全键位重绑定。
- 字幕。
- 字号可调。
- 色盲友好提示。
- 弹反窗口辅助。
- 受击伤害辅助。
- 镜头晃动强度。
- 长按 / 连点替代。
