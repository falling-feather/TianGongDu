## 2.2.1-f1.3｜F1 Web ABI 与 Guest 正常存档证据｜2026-07-12

### 已完成

- 冻结显式小端 Web ABI 1.0、`SaveEnvelopeV1`、Profile Head compare-and-swap 协调器与 256 KiB 分块异步存储桥；C++ 仍是快照和提交状态的唯一真相。
- 以机器合同建立独立 `prototype_f1/internal` IndexedDB v1：`profile_heads`、`snapshots`、`operations`、`profile_meta`、`device_settings`、`migration_workspace` 六个 object store 及复合键/索引。
- Web Shell 以固定 Guest Profile 启动；首次无 Head 时进入合法新档，保存只在 snapshot + Head 原子事务 `complete` 后增加 `committedSaveCount` 并显示“已保存”。

- ABI/IndexedDB 生成物、消息尺寸、128-bit ID、分块完成与空快照 UI 事件均有 Node/Native 合同测试；旧的几何 Canvas 仍只承担宿主生命周期探针。

- 根据项目负责人新决策，将正式平台改为浏览器首发的横版动作 RPG，不再采用 Godot/.NET 桌面优先方案。

- 新增机器可读技术基线与 Schema，并扩展测试守卫 C++、Axmol、Emscripten、WebGL2、DOM、IndexedDB 和同步决策。
