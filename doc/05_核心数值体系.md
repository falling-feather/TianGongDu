# 《天工渡 / Heavenwrights》核心数值体系

> 版本：0.3  
> 日期：2026-06-23  
> 状态：当前有效数值框架  
> 适用：Web 端 2D / 类 2.5D 像素动作 RPG 纵切。

## 1. 数值原则

- 动作优先，数值服务动作。
- 玩家主要靠观察、闪避、格挡、弹反、破绽处理和门径资源循环取胜。
- 成长优先给新动作、新解法、新资源循环，而不是单纯加攻击力。
- 战斗结果表达为“归位”，底层可以有生命值，表现层不强调击杀。

推荐成长比例：

- 70% 横向成长：新招式、新探索能力、新工艺特质、新 NPC 关系。
- 20% 小幅数值成长：气力、定力、恢复效率、资源上限。
- 10% 装备强化：器具特质、缺陷修复、材料差异。

## 2. 玩家核心属性

| 字段 | 中文名 | 作用 | 纵切建议 |
| --- | --- | --- | --- |
| `vitality` | 生机 | 玩家容错，归零后失败 / 撤退 / 救回 | 100 |
| `stamina` | 气力 | 闪避、重击、格挡维持、部分技艺消耗 | 100 |
| `poise` | 定力 | 抗打断能力 | 60 |
| `focus` | 专注 | 通用技艺资源 | 0-100 |
| `pathResource` | 门径资源 | 每个门径独立资源，伞师为风息 | 0-100 |
| `craftAffinity` | 匠心 | 工艺试炼与器具返修相关 | 0-5 阶 |

## 3. 伞师资源：风息

获得方式：

| 行为 | 获得风息 |
| --- | ---: |
| 普通格挡成功 | +4 |
| 完美格挡 | +10 |
| 弹反成功 | +18 |
| 雨中滑步经过风场 | +6 |
| 试伞工艺良好 | 战斗初始 +10 |

消耗方式：

| 技能 | 消耗 | 作用 |
| --- | ---: | --- |
| 收风 | 25 | 拉近小型敌人或打断远程 |
| 伞旋 | 35 | 范围控场，造成风压破绽 |
| 借风越行 | 持续消耗 | 探索回访和战斗换位 |
| 归伞反击 | 50 | 弹反后强化反击 |

伞师节奏：

```text
观察 -> 架伞 -> 完美格挡 / 弹反 -> 积风 -> 反击 / 控场 -> 换位
```

## 4. 敌人数值

| 字段 | 中文表达 | 作用 |
| --- | --- | --- |
| `disorder` | 失序值 | 相当于生命，归零后进入归位流程 |
| `stability` | 稳定 / 定力 | 被打空后暴露破绽 |
| `breakThreshold` | 破绽阈值 | 短时间破绽力超过阈值时硬直 |
| `guard` | 护势 | 敌人防御或结构保护 |
| `aggression` | 攻势倾向 | AI 行为频率 |
| `recovery` | 复稳速度 | 从破绽恢复的速度 |
| `weakTags` | 弱点标签 | 如 `wind`, `paper`, `fire`, `sound` |
| `resistTags` | 抗性标签 | 如 `rain`, `shadow`, `metal` |

纵切预算：

| 类型 | 失序值 | 稳定 | 目标时长 |
| --- | ---: | ---: | --- |
| 雨魇 | 45-60 | 30 | 8-15 秒 |
| 破伞影 | 70-90 | 45 | 15-25 秒 |
| 断骨伞灵 | 220-300 | 100 | 60-90 秒 |
| 雨巷破伞执念 Boss | 900-1200 | 分阶段 | 4-6 分钟 |

## 5. 战斗结算

伤害 / 归位进度：

```text
finalDamage =
  actionPower
  * pathwayModifier
  * artifactModifier
  * stateModifier
  * enemyVulnerability
  * difficultyModifier
```

破绽：

```text
breakDamage =
  actionBreakPower
  * timingBonus
  * postureBonus
  * weakTagBonus
```

规则：

- 不加入大随机波动。
- 暴击不作为主系统。
- 弱点命中和破绽窗口比随机暴击更适合本项目。
- Boss 破绽状态是机制窗口，不是长时间木桩。

## 6. 格挡与弹反

格挡：

```text
blockedDamage = incomingDamage * (1 - guardReduction)
staminaCost = incomingGuardPressure - guardPower
```

完美格挡：

- 普通难度窗口：180-220ms。
- 获得风息。
- 减少气力消耗。

弹反：

- 普通难度窗口：120-160ms。
- 造成大量破绽力。
- 解锁短反击窗口。

## 7. 工艺与器具特质

每件器具至少描述：

- `materialIds`
- `craftProcessIds`
- `makerNpcId`
- `qualityTier`
- `traits`
- `flaws`
- `visualVariant`
- `combatOverrides`
- `npcCommentIds`

示例：

| 特质 | 效果 |
| --- | --- |
| 竹骨匀称 | 格挡气力消耗 -8% |
| 伞面吃风 | 完美格挡风息 +2 |
| 纸性偏脆 | 连续格挡第三次后守势下降 |
| 蓝染稳色 | 湿滞抗性 +15% |

## 8. 地区四维

四维底层 0-100，对玩家显示等级和叙事反馈。

| 维度 | 含义 |
| --- | --- |
| 传承 | 师承是否延续，技艺是否失真，古法是否被理解 |
| 市场 | 当地产业是否能生存，商路和订单是否稳定 |
| 生活 | 技艺是否仍与普通人的生活有关 |
| 精神 | 技艺背后的信念、仪式和意义是否还在 |

首个纵切只做两次影响：

- 商业急单：市场提高，传承 / 精神可能降低。
- 完整工序：传承 / 精神提高，耗时或资源成本提高。

四维变化至少影响 NPC 台词、商店或材料、敌人失序强度、可回访事件、Boss 终局中的一项。
