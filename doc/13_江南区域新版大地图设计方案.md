## 0. 修改方案摘要

当前 07 号文档已经把江南从“首屏 demo”扩展成“第一版投放设计书”，但它的核心仍偏向纵切：围绕纸伞铺、竹林、夜市、旧桥和断桥 Boss 形成一条较清晰的 45-60 分钟体验。这个规模适合验证题材、操作和数据结构，但还不足以支撑一个正式章节。

新版江南应升级为一个不少于 5 小时的开放式地区，按四维共同推进：

| 维度 | 旧问题 | 新方案 |
| --- | --- | --- |
| 百工 | 纸伞工艺线存在，但多为一次性剧情/技能解锁 | 建立纸伞、染纸、竹作、舟修、桥工、灯市器具等可重复工艺循环 |
| 战斗 | 终局 Boss 强，精英和小 Boss 结构不足 | 设置 1 个核心 Boss、4 个小 Boss、8 类精英/核心任务敌人和可回访挑战 |
| 剧情 | 有主线但地区社会关系仍浅 | 以《天工录》首次入录为主线，串联师承、货账、旧桥、无款工印和玩家职分 |
| 探索 | 节点制纵切清楚，但自由探索和长期自给不足 | 做 6 个大型子区域，每区有 NPC、建筑、采集、工艺、战斗、支线和回访 |

新版不删除 07 号内容，而是把 07 号中的 11 个节点重组为 6 个大型子区域。每个子区域内部再包含多个点位。后续如果改版 07，建议以本文的 6 区大地图为骨架，将原 07 的节点作为区域内点位或主线关卡。

## 1. 章节规模

### 1.1 目标可玩时长

| 内容 | 目标时长 | 说明 |
| --- | ---: | --- |
| 主线剧情 | 1.5-2 小时 | 从水路入镇到旧桥归位，理解天工录与执灯渡人职分 |
| 百工探索 | 1-1.5 小时 | 纸伞、竹材、染纸、桐油、灯伞、舟修等工艺循环 |
| 战斗挑战 | 1-1.5 小时 | 小 Boss、精英战、巡检任务、旧桥回放 |
| 大地图探索 | 1-1.5 小时 | 自由采集、隐蔽点、屋檐路线、水路捷径、支线 NPC |
| 日常与回访 | 无限轻循环 | 订单、修伞、采材、跑单、试伞会挑战、资源刷新 |

M1 正式章节最低目标：首次通关 5 小时，完成 70% 支线约 7 小时，全收集和挑战约 10 小时。

### 1.2 自给自足循环

江南地区必须允许玩家在本区域长期游玩，不依赖其他地区补资源。

```text
采集竹材 / 纸料 / 桐油 / 染料 / 水产 / 香草
-> 作坊加工为伞骨、伞面、补片、油纸、灯伞、舟修材料
-> 用于战斗消耗、装备返修、NPC 委托和市场订单
-> 获得铜钱、师评、材料、工艺熟练度和四维回声
-> 解锁新采集点、支线、挑战、建筑服务和天工录条目
```

最低自给资源：

- 生存补给：雨伞补片、草药贴、简易饭食、水路鱼获。
- 工艺材料：青篾、韧竹、雨纹纸、桐油、蓝染料、麻线、风铃铆。
- 经济资源：夜市跑单、灯伞订单、舟修委托、桥工巡检、居民补伞。
- 战斗回收：雨魇残纸、旧伞骨、风息结、无款印拓片。

## 2. 四维推进结构

### 2.1 四条主推进线

| 推进线 | 章节主轴 | 玩家持续行为 | 主要产出 |
| --- | --- | --- | --- |
| 百工线 | 纸伞工艺从材料到试伞会 | 采集、加工、返修、订单、试艺 | 伞具特质、工艺等级、作坊服务 |
| 战斗线 | 雨季祟局从游祟到旧桥核心 | 巡检、清理、精英战、小 Boss、回放 | 技能节点、Boss 弱点、战斗素材 |
| 剧情线 | 玩家首次理解《天工录》如何入录 | 对话、调查、选择、落款、见证 | 主线推进、NPC 信任、无款工印 |
| 探索线 | 从水镇到竹林、外圩、屋檐、水路 | 自由行走、解锁捷径、发现点位 | 地图开放、隐藏资源、支线入口 |

### 2.2 四线互锁规则

- 百工线不能只在作坊里发生。采集、战斗掉落、NPC 委托和探索发现都要进入工艺。
- 战斗线不能只等最终 Boss。每个大型子区域至少有一个战斗问题：精英、小 Boss、巡检任务或祟局点。
- 剧情线不能只有主线 NPC。每个区域至少有两名 NPC 能提供当地生活、利益或记忆。
- 探索线不能只给宝箱。每个区域至少有一种工艺资源、一条捷径或一个回访变化。

## 3. 江南大地图总览

新版江南地图设计 6 个大型子区域，每个大型子区域内部有 6-12 个可交互点位。玩家从烟雨竹巷进入，但开场 30 分钟后可以较自由地去河埠水市、青篁湿林或染纸晒场；风铃旧桥和湖田外圩属于中高风险区域，既可主线推进，也可提前探索。

| 序 | 大区 | 主要维度 | 风格 | 预估内容量 |
| ---: | --- | --- | --- | ---: |
| 1 | 烟雨竹巷 | 百工 / 剧情 | 多建筑、多 NPC、多雨棚、纸伞作坊群 | 60-80 分钟 |
| 2 | 河埠水市 | 探索 / 剧情 / 市场 | 水路、码头、夜市、船队、账棚 | 50-70 分钟 |
| 3 | 青篁湿林 | 百工 / 探索 / 战斗 | 竹林、湿地、山风、采集和伏击 | 45-70 分钟 |
| 4 | 染纸晒场 | 百工 / 剧情 / 机关 | 染缸、晾纸、风燥棚、湿气机关 | 45-60 分钟 |
| 5 | 风铃旧桥 | 战斗 / 剧情 | 旧桥、风铃祠、桥洞、雨幕裂口 | 70-90 分钟 |
| 6 | 湖田外圩 | 探索 / 自给 / 战斗 | 圩田、芦苇、水闸、舟修和野外委托 | 60-90 分钟 |

核心 Boss 位于风铃旧桥深层的“断桥雨幕”。四个小 Boss 分布于河埠水市、青篁湿林、染纸晒场和湖田外圩。烟雨竹巷主要承担剧情、工艺和城市战斗教学。

## 4. 大区一：烟雨竹巷

### 4.1 区域定位

烟雨竹巷是江南地区的主城镇和纸伞工艺核心区。这里不是一条装饰性街巷，而是玩家的地区基地：学艺、返修、接单、换伞骨、对话、天工录初始说明都发生在这里。

地形特征：

- 多建筑、多 NPC、多雨棚、少耕地。
- 白墙黛瓦、窄巷、连廊、雨棚和晒伞架形成密集动线。
- 可自由进入多个工坊小院，建筑点位密度最高。
- 屋檐上有隐藏路线，后期可用借风越行串联主巷和旧桥。

### 4.2 百工玩法

| 工艺点 | 玩法 | 产出 |
| --- | --- | --- |
| 伞骨换装 | 选择青篾、老竹、风折竹，替换伞骨 | 耐久、格挡消耗、伞旋稳定 |
| 伞面修补 | 用雨纹纸、油纸、蓝染纸修补伞面 | 湿滞抗性、风息收益、外观 |
| 试伞架 | 在小风口做开合测试 | 新技能节点和师评 |
| 居民补伞 | 日常委托，消耗低级材料 | 铜钱、生活维度、居民台词 |
| 工艺手账 | 记录工序和材料来源 | 天工录条目、资料核验标记 |

### 4.3 NPC

| NPC | 描述 | 交互方式 |
| --- | --- | --- |
| 莫鸣 | 30 岁左右的纸伞工艺传承者，外表温和但手上工夫很硬，负责把“伞是器具不是装饰”的观念讲清楚。 | 提供材料锻造、更换伞骨、提高伞具耐久度；可检查玩家伞具缺陷并给返修建议。 |
| 沈雨 | 老伞师，守完整试伞规矩，也是玩家认识天工录“师承记录”的第一位关键见证人。 | 教架伞、评工序、开启师承试炼；根据玩家选择给最终评价。 |
| 阿青 | 作坊学徒，急着证明快工序也能做出好伞。 | 陪练、试验新工序、触发“捷径是否可靠”支线。 |
| 乔针 | 缝匠，擅长穿线和补边，嘴快但靠谱。 | 修复线脚缺陷，降低伞旋不稳概率。 |
| 阿棠 | 巷口住户，关心日用伞和邻里生活。 | 发起居民补伞、雨棚漏水、老人出行等生活委托。 |
| 姚记档 | 天工院守档吏，负责解释《天工录》不是图鉴，而是活态工艺记录。 | 解锁天工录页面，记录地区选择、工序和无款工印线索。 |

### 4.4 建筑

| 建筑 | 功能 |
| --- | --- |
| 莫家伞坊 | 主工艺台：换伞骨、返修、耐久提升。 |
| 沈雨旧铺 | 师承教学、门径技能、剧情对话。 |
| 连檐雨棚 | 居民委托板、天气公告、低压社交。 |
| 伞骨仓 | 存放材料，可被雨魇扰动，触发小战斗。 |
| 天工录案桌 | 章节记录、工艺条目、主线回顾。 |
| 屋檐小道 | 后期借风越行路线，通向水市和旧桥上层。 |

### 4.5 采集点

- 旧伞架：旧伞骨、破线、残纸。
- 雨棚滴水槽：雨水样本，用于试伞。
- 伞骨仓角落：青篾碎料、麻线。
- 屋檐鸟巢旁：风铃铆碎片。
- 居民门口：日用伞委托和小额材料回收。

### 4.6 战斗内容

| 内容 | 说明 |
| --- | --- |
| 游祟清扫 | 低强度雨魇和破伞影，服务入门战斗。 |
| 作坊保卫 | 材料仓被雨幕扰动，玩家在狭窄空间练架伞。 |
| 学徒切磋 | 阿青陪练，可反复挑战，产出少量风息熟练度。 |
| 精英敌人：线脚乱影 | 出现在伞骨仓，攻击前摇明显，教玩家弹反。 |

### 4.7 长期循环

- 每日补伞委托刷新 3-5 个。
- 作坊根据材料库存提供不同伞骨升级。
- 居民台词随生活维度变化。
- 若市场维度高，订单多但作坊更忙；若传承高，师傅提供更稳定的工序加成。

## 5. 大区二：河埠水市

### 5.1 区域定位

河埠水市是江南的经济、交通和夜市核心区。这里让玩家理解“市场”不是坏东西，而是工艺能活下去的现实条件。区域白天偏码头与账棚，夜晚变成灯市和货摊。

地形特征：

- 多水路、多船只、多摊位、桥上桥下双层动线。
- 白天有货箱、船队和账房；夜晚有灯伞、食摊、跑单路线。
- 水道可自由摆渡，后期变成快速旅行网络。

### 5.2 百工玩法

| 工艺点 | 玩法 | 产出 |
| --- | --- | --- |
| 灯伞装配 | 将纸伞与灯架结合 | 夜市声望、精神维度、视觉变体 |
| 货箱防潮 | 用油纸和竹框保护材料 | 市场收入、材料保存 |
| 舟篷修补 | 修船篷和货篷 | 水路快捷、船夫好感 |
| 跑单分配 | 选择订单优先级 | 四维变化、账册记录 |

### 5.3 NPC

| NPC | 描述 | 交互方式 |
| --- | --- | --- |
| 骆青禾 | 夜市跑单人，精明又辛苦，代表货期压力。 | 发布急单、跑单任务、市场选择。 |
| 房掌 | 行会账房，重视货账和信用。 | 查账、订单排序、账页纠偏支线。 |
| 柳江 | 船夫，负责水路摆渡和水势信息。 | 开启水路快捷、护送任务、捕鱼点位提示。 |
| 蓝文 | 染纸艺人，在夜市展示灯伞花色。 | 改伞面视觉，接灯伞订单。 |
| 郑小满 | 年轻货栈伙计，知道各家商队传闻。 | 提供传闻、隐藏委托和外地订单线索。 |
| 水市阿婆 | 卖热汤和补给，记得很多旧桥故事。 | 购买低价补给，解锁旧桥传闻。 |

### 5.4 建筑

| 建筑 | 功能 |
| --- | --- |
| 水市码头 | 船只出入、摆渡、护送入口。 |
| 夜市账棚 | 订单选择、市场四维结算。 |
| 灯伞台 | 灯伞制作、夜市表现和精神维度反馈。 |
| 货栈防潮仓 | 材料存储、防潮小游戏和潜在战斗。 |
| 舟修棚 | 舟篷修补、快速旅行升级。 |
| 食摊长廊 | 恢复、传闻、生活支线。 |

### 5.5 采集点

- 码头绳桩：麻绳、湿木楔。
- 货箱角落：防潮纸、破损账页。
- 船篷残片：油布、竹篾。
- 夜市灯架：灯纸、铜钩。
- 水面漂流物：雨纹纸、残伞柄。

### 5.6 战斗内容

| 内容 | 说明 |
| --- | --- |
| 小 Boss：潮锚账影 | 夜市账册和潮锚祟局结合的小 Boss，攻击会拉扯玩家位置。 |
| 码头护送 | 防守货箱，保护桐油和伞骨不被雨魇拖走。 |
| 夜市突发 | 人群中出现雨账影，要求不伤 NPC 的短战斗。 |
| 船队巡检 | 水上小型战斗，利用架伞挡雨针。 |

### 5.7 长期循环

- 夜市订单每日刷新，分普通、急单、信誉单。
- 水路摆渡随舟修等级开放更多停靠点。
- 市场维度高会提高收入，但也提高急单频率。
- 精神维度高会增加灯市活动和风铃线索。

## 6. 大区三：青篁湿林

### 6.1 区域定位

青篁湿林是江南百工资源的主要来源区，也是自由探索感最强的野外区域。它承担采竹、找替代材料、地形探索和野外精英战。

地形特征：

- 多竹林、多湿地、多高低差，建筑少但点位分散。
- 有竹桥、溪沟、风折竹、临时工棚和废弃竹楼。
- 雨后会出现新的水路和可采集点。

### 6.2 百工玩法

| 工艺点 | 玩法 | 产出 |
| --- | --- | --- |
| 观竹节 | 用工灯看竹节纹理 | 材料评级 |
| 选竹龄 | 老竹、快干竹、风折竹三类选择 | 不同伞骨特质 |
| 竹篾处理 | 切篾、晾篾、捆扎 | 伞骨耐久和重量 |
| 竹器副产 | 制作竹筐、竹桥补件 | 建筑修复、生活委托 |

### 6.3 NPC

| NPC | 描述 | 交互方式 |
| --- | --- | --- |
| 周桐 | 竹林采工，讲实用但不愿糟蹋竹材。 | 采竹教学、替代材料、折中路线。 |
| 段婆 | 看林老人，熟悉雨季风口。 | 告诉玩家天气和采集刷新点。 |
| 林九 | 年轻猎户，常巡湿林边缘。 | 发现精英敌人、野外挑战和陷阱点。 |
| 小墨 | 迷路学徒，偷偷来找快干竹。 | 支线：带回作坊或让他完成试采。 |
| 竹楼匠 | 修临时竹桥的人，和水市有材料交易。 | 修桥、开启捷径、交换竹器。 |

### 6.4 建筑

| 建筑 | 功能 |
| --- | --- |
| 竹料栈 | 材料交易和采竹任务入口。 |
| 临溪切竹台 | 处理竹篾，制作伞骨半成品。 |
| 风折竹坡 | 高风险采集点，产出折中结局材料。 |
| 竹桥工棚 | 修桥、开捷径、野外休息。 |
| 废竹楼 | 精英敌人巢点和隐藏天工录碎页。 |

### 6.5 采集点

- 老竹丛：高耐久伞骨材料。
- 快干竹棚：市场路线材料。
- 风折竹坡：折中路线关键材料。
- 溪边苔石：草药、雨水样本。
- 废竹楼：旧篾、风铃铆、无款印拓片。

### 6.6 战斗内容

| 内容 | 说明 |
| --- | --- |
| 小 Boss：青篁裂骨影 | 竹骨祟局小 Boss，弱收风和弹反，击败后开放高处竹桥。 |
| 精英：竹壳影 | 位移快，逼玩家利用地形和收风。 |

- 采集过度会降低传承和精神反馈，合理轮采可提升生活。
- 玩家可长期生产伞骨、竹器和桥修材料。

## 7. 大区四：染纸晒场

染纸晒场负责把纸伞从“骨架”推进到“可用器具”。这里是百工线的第二核心，同时承担视觉风格、状态抗性和湿气机关。

地形特征：

## 8. 大区五：风铃旧桥

## 11. 剧情主线新版

### 11.1 主线主题

江南是玩家第一次真正理解《天工录》的地区。《天工录》不是打卡式图鉴，也不是纯粹文化百科，而是记录一门工艺如何在人、器、市场、生活和祟局中仍然活着。

主线问题：

> 一把伞怎样才算“可入录”？是货期保住、工序完整、居民有伞可用，还是旧桥雨幕真正归位？

### 11.2 主线七章

| ---: | --- | --- | --- | --- |

| 4 | 夜市有账 | 河埠水市 | 急单、灯市、居民日用发生冲突。 | 剧情、探索 |
| 5 | 外圩涨水 | 湖田外圩 | 水闸和材料供应出问题，显示地区自给与民生。 | 探索、百工 |

### 12.1 地图开放方式

- 未修桥梁。
