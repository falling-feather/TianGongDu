# Heavenwrights Web Demo

> 文档定位：说明当前浏览器原型的运行方式、范围和操作。  
> 适用范围：`prototype/web-demo/` 的无外部依赖 Canvas 原型。  
> 最后更新：2026-07-10
> 维护者：原型维护者  
> 关联文档：`../../doc/03_项目全局设计规划.md`、`../../doc/07_江南雨巷纵切规格.md`、`../../doc/12_测试与验收指南.md`

无外部依赖的浏览器原型，用于验证“江南雨巷 + 纸伞门径”的短闭环体验。归档版本已补齐阶段路线、自然玩法触发、真实失败、据点重试、深层菜单暂停、Boss 归位和《天工录》完成落款。

运行：

```bash
npm run dev
```

打开：

```text
http://127.0.0.1:4174/
```

操作：

- `WASD` / 方向键：移动
- `Shift` / `Space`：冲刺
- `Q`：架伞
- `E`：收风
- `R`：借风越行
- `F`：交互 / 归位
- `M`：地图
- `I`：背包
- `N`：人物
- `T`：小憩推进时辰

完成路线：

```text
入镇与架伞 -> 采竹与伞面工艺 -> 夜市与风铃铆
-> 旧桥战斗 -> 断桥 Boss 归位 -> 返回纸伞铺落款
```

普通入口默认隐藏“推进 M1 流程”白盒按钮。需要快速验证内容数据时，使用：

```text
http://127.0.0.1:4174/?debug=1
```

生机归零后可从阶段据点重新整备，已完成主线、材料与选择不会丢失。背包、人物、大地图、编辑器、江南 M1 和记录面板打开时会暂停战斗。

验收路径见 `../../doc/12_测试与验收指南.md`。本 demo 是 M0 原型，不替代最终 Axmol/C++ 实现。
