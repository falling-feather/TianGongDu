# 天工渡旧版归档分支

本分支完整封存第一代 Web/Canvas 项目。所有可运行代码、内容数据、文档、测试与工具已收纳到：

[`legacy/Heavenwrights-Web-V1.1.0/`](legacy/Heavenwrights-Web-V1.1.0/)

归档版本：`V1.1.0 · 江南雨巷阶段可玩节点`。

运行：

```bash
cd legacy/Heavenwrights-Web-V1.1.0
npm test
npm run validate:content
npm run dev
```

默认地址：`http://127.0.0.1:4174/`。开发流程捷径使用 `http://127.0.0.1:4174/?debug=1`。

本分支只用于历史回看、验证与比较，不接受新版生产代码。新版世界观、架构和 1.0 设计在主线分支继续推进。
